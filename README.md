<h2>Sobre</h2>
É um web service, puro consumido via curl. Elaboramos uma locadora de jogos, filmes e series.<br>
O sistema esta RestFul, possui as quatro operações: GET, POST, PUT, DELETE.

======================================================================

<p>Tabela Jogos</p>
```

```
